Integration Instructions

  1. Include our script in your HTML:
    <script src="https://your-domain.com/widget.js"></script>
  2. Initialize the widget with your API key:
    const chatbot = new ChatbotWidget({ apiKey: 'YOUR_API_KEY', position: 'bottom-right', primaryColor: '#007bff', title: 'Chat with AI', placeholder: 'Type your message...' });
  3. Customize the appearance and behavior using the available options:
    // Available options { apiKey: string, // Your API key (required) position: string, // 'bottom-right' or 'bottom-left' primaryColor: string, // Any valid CSS color title: string, // Chat widget title placeholder: string, // Input placeholder text width: number, // Widget width in pixels height: number // Widget height in pixels }
