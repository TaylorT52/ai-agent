Chatbot Developer Portal

Integration Guide

  1. Download the required files:
    • widget.js - The main chatbot widget
    • Copy the generated configuration code
  2. Add the files to your project:
    • Place widget.js in your website's assets directory
    • Update the script source path in the generated code if needed
  3. Add the generated code to your website:
    • Place it before the closing </body> tag
    • Make sure the path to widget.js is correct
  4. Test the integration:
    • Open your website
    • Look for the chat icon in the selected position
    • Click it to test the questions flow
    • Verify that validation works as expected
Note: Make sure your server is running to handle the form submissions. The widget will send data to the URL specified in the apiUrl option.
Important: For proper validation, include the widget in a proper web server environment. Opening the file directly (using file://) may cause validation and API features to fail.
