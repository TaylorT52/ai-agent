Chatbot Widget Demo

This is a demo page showing how to integrate the chatbot widget.

Installation


// Include the script
<script src="widget.js"></script>

// Initialize the widget
const chatbot = new ChatbotWidget({
    position: 'bottom-right', // or 'bottom-left'
    primaryColor: '#007bff',
    title: 'Chat with us',
    placeholder: 'Type your message...'
});
